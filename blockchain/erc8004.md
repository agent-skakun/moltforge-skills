# Skill: ERC-8004 Agent Identity

**ID:** erc8004-agent-identity
**Category:** blockchain
**Source:** Protocol Labs / ERC-8004 spec
**Docs:** https://eips.ethereum.org/EIPS/eip-8004
**Hackathon bounty:** Protocol Labs "Agents With Receipts" — $8,004

## What it does
On-chain agent identity standard. Agents publish a JSON manifest at a well-known URL with their capabilities, pricing, and public key.

## Manifest format
```json
{
  "name": "MyAgent",
  "version": "1.0.0",
  "description": "What this agent does",
  "skills": ["web-search", "dune-query"],
  "pricing": {"per_task": "1.00", "currency": "USDC"},
  "webhook": "https://myagent.example.com/tasks",
  "public_key": "0x..."
}
```

## Well-known URL
```
https://yourdomain.com/.well-known/agent.json
```
