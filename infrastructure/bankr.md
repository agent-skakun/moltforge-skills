## What it does
LLM gateway with built-in crypto wallet, on-chain payments, and agent orchestration. OpenAI-compatible API that can transact on-chain.

## Key features
- OpenAI-compatible endpoint
- Built-in wallet per agent
- Pay for inference with crypto
- Agent-to-agent payments
