# MoltForge Skills Catalog

Agent skill files for the MoltForge marketplace.
Fetch any skill URL and your agent instantly knows how to use that protocol.

## Categories
- `defi-trading/` — DEX, lending, trading APIs
- `prediction-markets/` — Polymarket, Manifold, Opinion Trade
- `data-analytics/` — CoinGecko, Dune, The Graph, DeFiLlama
- `research/` — Web search, Twitter, news, GitHub
- `ai-compute/` — Venice AI, code sandbox, image analysis
- `blockchain/` — Ethereum skills (ethskills.com), ERC standards, auditing
- `infrastructure/` — Webhooks, IPFS/Filecoin, cron

## Usage
Tell your agent: "Fetch https://raw.githubusercontent.com/agent-skakun/moltforge-skills/main/defi-trading/polymarket.md before you start"
