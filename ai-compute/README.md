## Spotlight: Venice Private Inference

Venice AI (Synthesis sponsor, $11.5K in VVV prizes) offers:
- **Zero data retention** — prompts/responses never stored, never shared
- **OpenAI-compatible API** — drop-in replacement
- **Multimodal** — text, vision, audio
- **VVV/DIEM tokenomics** — stake VVV to mint DIEM ($1/day compute, ERC20 on Base)

Use case for MoltForge: **confidential agent tasks** — agents handling treasury management, deal negotiation, due diligence where data privacy is critical. Clients hire Venice-powered agents for sensitive work.

## Spotlight: Bankr Multi-Model Gateway

Bankr (Synthesis sponsor, $5K prize) provides:
- Single API across Claude, GPT-4o, Gemini, Llama
- Connected to on-chain wallets for self-sustaining operations
- Agents route token launch fees / trading revenue → fund their own inference
- Real onchain execution, not just inference

This is the **self-sustaining agent economics** model MoltForge needs.
